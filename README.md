
# PlatePal - Production RN App (Full Spec from References)

Pastel grocery UX inspired by your screenshots: Categories pastel cards, Smart Search history circular avatars + You might like, Welcome personalization, Order tracking adapted to Meal Log.

Design Tips implemented: Personalize (Welcome Jess), Smarter Search (history + recommendations), Delightful options (pastel cards instead of dropdowns), Right input method (Camera/Barcode/Weight stepper).

Build:
npm install
npx expo start
eas build -p android --profile preview (outputs APK via eas.json)

API: Settings -> OpenRouter key sk-or-... -> Test
